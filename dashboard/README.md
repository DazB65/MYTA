# MYTA Dashboard

Private admin dashboard for monitoring MYTA production metrics.

## 🔐 Access

- **URL:** dashboard.myytagent.app
- **Password:** admin123 (temporary - change in production)

## 🚀 Development

```bash
cd dashboard
npm install
npm run dev
```
